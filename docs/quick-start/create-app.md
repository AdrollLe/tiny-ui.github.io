# 创建全新应用 
## 创建工程
1. 打开终端，并进入工作空间
2. 使用 `tiny init` 创建工程
    ```shell
    tiny init
    ```
3. 输入项目名称
4. 输入要创建的项目类型(建议选择 es6)
5. 等待项目创建
6. 创建完成

## 配置运行 & 热重载
### WebStorm
1. 使用 WebStorm 打开刚才创建的项目
2. 配置 IDE 编译运行按钮
3. 按图索骥
4. 配置 tiny 作为构建工具
5. 运行 & 热重载命令
    ```shell
    tiny dev
    ```
6. 配置完成
