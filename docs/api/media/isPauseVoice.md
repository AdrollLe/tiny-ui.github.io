# isPauseVoice
## 说明
查询当前语音是否暂停播放

## 示例
```javascript
var result = TinyAPI.media.isPauseVoice();
```

## 出参

| 属性名    | 类型      | 属性说明 | 必填  | 默认值     | 取值范围                 |
|:-------|:--------|:-----|:----|:--------|:---------------------|
| result | boolean | 查询结果 |     |  |  |
