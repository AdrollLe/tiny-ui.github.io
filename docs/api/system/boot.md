# boot
## 说明
是否设置应用开机自启

## 示例
```javascript
TinyAPI.system.bootListener(true);
```

## 参数

| 属性名      | 类型      | 属性说明   | 必填  | 默认值   | 取值范围                                                         |
|:---------|:--------|:-------|:----|:------|:-------------------------------------------------------------|
| enable   | boolean | 是否开机自启 | 是   | false |                                                              |
