# full screen
## 说明
全屏模式

## 示例
```javascript
TinyAPI.interact.fullScreen(true)
```

## 参数

| 属性     | 类型      | 属性说明     | 必填  | 默认值   | 取值范围   |
|:-------|:--------|:---------|:----|:------|:-------|
| enable | boolean | 是否开启全屏模式 | 是   | false |  |
