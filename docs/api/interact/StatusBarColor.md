# status-bar color
## 说明
设置顶部状态栏颜色

## 示例
```javascript
TinyAPI.interact.statusBarColor("#FFFFFF")
```

## 参数

| 属性    | 类型     | 属性说明   | 必填  | 默认值 | 取值范围   |
|:------|:-------|:-------|:----|:----|:-------|
| color | string | 状态栏颜色值 | 是   |   |  |
