# status-bar height
## 说明
获取顶部状态栏高度

## 示例
```javascript
var height = TinyAPI.interact.statusBarHeight()
// height = 22
```

## 出参

| 属性     | 类型     | 属性说明  | 必填  | 默认值 | 取值范围   |
|:-------|:-------|:------|:----|:----|:-------|
| height | number | 状态栏高度 |     |   |  |
