# navigation-bar color
## 说明
设置底部导航栏颜色

## 示例
```javascript
TinyAPI.interact.navigationBarColor("#FFFFFF")
```

## 参数

| 属性    | 类型     | 属性说明   | 必填  | 默认值 | 取值范围   |
|:------|:-------|:-------|:----|:----|:-------|
| color | string | 导航栏颜色值 | 是   |   |  |
