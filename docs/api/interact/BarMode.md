# bar mode
## 说明
状态栏反白模式

## 示例
```javascript
TinyAPI.interact.statusBarLightMode(true)
```

## 参数

| 属性   | 类型      | 属性说明  | 必填  | 默认值   | 取值范围   |
|:-----|:--------|:------|:----|:------|:-------|
| mode | boolean | 状态栏反白 | 是   | false |  |
