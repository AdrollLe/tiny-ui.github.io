# navigation-bar height
## 说明
获取底部导航栏高度

## 示例
```javascript
var height = TinyAPI.interact.navigationBarHeight()
// height = 40
```

## 出参

| 属性     | 类型     | 属性说明  | 必填  | 默认值 | 取值范围   |
|:-------|:-------|:------|:----|:----|:-------|
| height | number | 导航栏高度 |     |   |  |
